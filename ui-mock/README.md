# UnicornMarket UI Mock

Static mock UI for review (no backend, no wallet).

## Open locally
- Double click `index.html`
- or run:

```bash
cd unicorn-market/ui-mock
python3 -m http.server 8080
# open http://localhost:8080
```

## Notes
- Mobile-friendly tab bar included
- Desktop 3-column terminal layout
- Mock data only
- Canonical default market: Unicorn (unwrapped) <-> wMEAT
